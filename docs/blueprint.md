# **App Name**: NeonTune

## Core Features:

- YouTube Music Integration: Seamlessly connects to the user's YouTube Music account to access their music library and playlists.
- Listening History Analysis: Analyzes the user's YouTube Music listening history to build a detailed profile of their music tastes.
- AI-Powered Playlist Curation: Uses the listening history profile as a tool to curate personalized playlists that match the user's preferences.
- Real-time Request System: Allows users to submit song requests, which are then added to a dynamic queue.
- Dynamic Queue Management: Manages the order of songs in the queue, prioritizing requests and blending them with AI-generated suggestions.
- User Profile: A dedicated user profile for submitting and managing music requests, enabling enhanced communication and control within the platform.

## Style Guidelines:

- Primary color: Electric purple (#BE29EC) for a modern, vibrant feel, reflecting the user's eclectic music taste.
- Background color: Dark grey (#222222) to create a sleek and immersive experience.
- Accent color: Neon green (#39FF14) to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk', a sans-serif font for titles and prominent text.
- Body font: 'Inter', a sans-serif font for the main content and descriptions.
- Use neon-style icons to represent different music genres, controls, and features.
- Subtle animations on song transitions and interactive elements.
